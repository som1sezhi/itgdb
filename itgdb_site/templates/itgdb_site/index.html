What is ITGDb?

ITGDb is a database for In The Groove-style StepMania charts. It aims to make it easier than ever to find charts to your favorite songs, or just to discover charts in general.

The source code for this website can be found on GitHub.

If you have any inquiries/comments/ideas/whatever concerning this website, feel free to contact me at @som1sezhi on Discord.
